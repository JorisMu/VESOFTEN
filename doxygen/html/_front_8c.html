Detailed Description

VGA_core DMA LIB 320x240, 8bit color.

Deze C-code draait op een STM32F4 microcontroller en implementeert een UART-command interface waarmee je via tekstcommando’s (bijv. via een terminal) grafische opdrachten kunt sturen naar een VGA/logic-laag.
