Detailed Description

Front-end interface voor de VGA-aansturing op de STM32F4.

Author
JB (mods by J. de Bruijne)

Deze module verzorgt de communicatie (UART), het parseren van tekstuele commando's en het doorsturen daarvan naar de logische laag.

Version
1.0
Date
2026-01-08
Note
Gebruikt de VGA_core DMA bibliotheek voor 320x240 resolutie met 8-bit kleur.
