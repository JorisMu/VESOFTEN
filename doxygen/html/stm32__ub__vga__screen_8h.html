Detailed Description

VGA-Screen-Library for STM32F4xx.

This library offers functions to control a 320x240 pixel VGA screen. It uses TIM1 for the pixel clock, TIM2 for HSync, and DMA2 for transferring pixel data to the GPIO pins.

Date
05.01.2026
Author
J. Mullink
Note
This library is based on an existing open-source project and has been adapted for this specific application.
