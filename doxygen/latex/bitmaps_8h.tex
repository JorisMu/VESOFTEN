\doxysubsection{Detailed Description}
Declarations for bitmap graphics. 

This file defines the data structures, identifiers (IDs), and external data array for bitmap images used in the application. The actual bitmap data is defined in {\ttfamily \doxylink{bitmaps_8c}{bitmaps.\+c}}.

A special color value used to represent a transparent pixel in a bitmap. 

When drawing a bitmap, pixels with this color value will not be drawn to the screen, allowing the background to show through.

Extern declaration for the global array of bitmaps. 

This array holds all the {\ttfamily \doxylink{struct_bitmap__t}{Bitmap\+\_\+t}} structures. The actual definition and initialization of this array is in {\ttfamily \doxylink{bitmaps_8c}{bitmaps.\+c}}.

Extern declaration for the global array of bitmaps.

This array collects all bitmap structures, making them accessible to the drawing functions via their index (which corresponds to the \doxylink{bitmaps_8h_a41164408d521f8ad3691d6854c5a7f76}{BITMAP\+\_\+\+ID} enum).
