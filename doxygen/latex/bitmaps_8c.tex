\doxysubsection{Detailed Description}
Bitmap pixel data definitions. 

This file contains the actual raw pixel data for the bitmaps that are declared in {\ttfamily \doxylink{bitmaps_8h}{bitmaps.\+h}}. The data is organized into 2D arrays, where each value represents an 8-\/bit color.
