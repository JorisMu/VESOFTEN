Font definitions for the VGA library. 

This file provides the data and structures for rendering text. It includes two fonts:\+
\begin{DoxyEnumerate}
\item Consolas:\+ A 5x7 monospaced font for a classic, fixed-\/width look.
\item Arial:\+ A proportional font for more natural-\/looking text.
\end{DoxyEnumerate}
