\doxysubsection{Detailed Description}
Bitmap pixel data definitions. 

Logic layer voor VGA-\/tekencommando’s. Deze laag valideert invoer, vertaalt parameters naar VGA-\/codes en vormt de tussenlaag tussen applicatie en VGA-\/driver.
